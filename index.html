EXAMPLES OF MY WORK

Below you can see a few samples of the work I have done together with a short description of the project, it's requirements and a view of the project. To see the code for the project on GitHub, just press the image!

Spotify

This project consisted of creating what would look like a Spotify music player. I was told to open the Spotify music player on my laptop and to recreate the layout from scratch without any design guidelines. This was my first attempt at creating something from scratch without having the guidelines from Figma to help along the way.

Patagonia

This project focused on making a site responsive to different screen sizes. I was given the design in Figma with three different sizes (desktop, tablet, and mobile). I then recreated the design from scratch and implemented the necessary flexible code and media queries to make the website look good not only on the three different device screen sizes, but on all sizes inbetween as well.
